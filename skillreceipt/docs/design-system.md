# Tailwind Design System (SkillReceipt)

## Visual Direction
High-trust, minimalist fintech UI inspired by modern payment platforms like Stripe and PayPal. Clean, calm, enterprise-grade—not crypto or flashy fintech.

---

## Color System (IMPLEMENTED VIA TAILWIND + CSS VARIABLES)

### Base Colors
- Background: `#FFFFFF`
- Surface: `#F8FAFC`
- Surface Alt: `#F1F5F9`
- Border: `#E2E8F0`
- Text Primary: `#0F172A`
- Text Muted: `#64748B`

### Brand Colors
- Primary (Actions): `#2563EB`
- Primary Dark: `#1D4ED8`
- Accent (Optional UI highlights): `#4F46E5`

### Status Colors
- Success: `#10B981`
- Warning: `#F59E0B`
- Error: `#EF4444`

---

## Typography Scale
- Display: 40–56px, semibold, tight tracking
- H1: 32px, semibold
- H2: 24px, semibold
- H3: 20px, semibold
- Body: 16px, regular
- Small: 14px
- Micro: 12px

---

## Spacing System
- Base unit: 8px
- Common spacing: 8 / 16 / 24 / 32 / 40 / 48 / 56 / 64
- Section padding: 24–32px
- Card padding: 24–32px
- Layout gutter: 16px mobile / 24px desktop

---

## Button System

### Primary Button
- Filled blue
- Used for: main actions, CTAs

### Secondary Button
- White background + blue border
- Used for: navigation actions, alternatives

### Ghost Button
- Transparent with hover background
- Used for: nav links, low emphasis actions

---

## Card System
- Background: white or `#F8FAFC`
- Border: `#E2E8F0`
- Shadow: subtle only (or none preferred)
- Radius: large (12–16px)
- Keep spacing airy and consistent

---

## Layout Guidelines

### Landing Page
- Hero section: centered, high contrast
- Features: grid-based, minimal noise
- How it works: step-based clarity
- Business value section: slightly darker background for emphasis
- FAQ: collapsible cards

### Dashboard
- Sidebar (desktop), top nav (mobile)
- Summary cards at top
- Lists in middle
- Details at bottom
- Wallet/status top-right
